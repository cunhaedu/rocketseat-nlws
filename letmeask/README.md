## :construction_worker: Getting started

Clone the project and access the folder.

```
$ git clone https://github.com/cunhaedu/letmeask.git
$ cd letmeask
```

Follow the steps below:

```
# Install the dependencies
$ yarn

# Start the project
$ yarn start
```
